# Keys [![version-badge]][version-link] [![license-badge]][MIT-link] [![pypi-badge]][pypi-link] ![issues-badge] ![fork-badge] ![start-badge]



`keys`是一个简单的密码管理器


### 示例


### 使用方式

```
usage: 
keys-config # 编辑密码
keys # 启动管理器
ctrl + k # 唤醒GUI界面
```


### 安装

```
$ pip install keys-manager
```
